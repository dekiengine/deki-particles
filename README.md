# Deki Particles

Modular particle system for the Deki Engine: emission shapes (point/circle/rect), gravity, drag, initial velocity, lifetime/size/color/rotation modifiers.

Part of the [Deki Engine](https://github.com/Kirbyrawr/deki-engine) module ecosystem.

## Installation

Install via the Module Manager inside the Deki Editor.

## Dependencies

| Dependency | Type |
|---|---|
| `deki-rendering` | Deki module |
| `deki-2d` | Deki module |
